## [1.0.0-beta] - 2026-03-23

### Added
- Core SQLite encrypted vault database using AES-256-GCM.
- Zero-knowledge authentication model with scrypt KDF.
- Fastify API backend and React/Vite frontend dashboard.
- Playwright automation engine for automated password rotations.
- Mobile network sync using single-use QR session tokens.
- PDFKit integration for Emergency Recovery Kit generation.
- Browser CSV password import and mapping system.
- Electron wrapper for local desktop runtime and packaging.
