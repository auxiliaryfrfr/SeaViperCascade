# Security Policy

## Supported Versions
Currently, only the latest release of SVC is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |


## Reporting a Vulnerability

**Please do not report security vulnerabilities through GitHub issues.**

If you discover a security vulnerability within SVC, please send an email to **auxiliaryfrfr@gmail.com**.

All security vulnerabilities are taken seriously and will be worked on to patch and release a fix as quickly as possible.
